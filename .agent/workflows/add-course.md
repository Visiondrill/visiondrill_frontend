---
description: How to add a new course as an instructor
---

This workflow guides an instructor through the process of creating a new instructional asset (course) on Visiondrill.

### Step 1: Initialize Course Creation
1. Navigate to the **Instructor Dashboard** (`/instructor`).
2. Select **My courses** from the sidebar or click the **Plus** icon in the Command Search.
3. Click the **Create module** button to open the multi-step creation flow.

### Step 2: Define Module Identity
1. **Title:** Enter a descriptive title for your course (e.g., "Advanced System Architecture").
2. **Vertical:** Select the category that best fits your content (Technology, Business, or Creative).
3. Click **Infrastructure settings** to proceed.

### Step 3: Configure Monetization
1. **Valuation:** Enter the price in KES. Set to 0 if the module is intended to be a free asset.
2. Review the monetization tip for performance insights.
3. Click **Finalize architecture** to proceed.

### Step 4: Deployment
1. Review the summary of your configuration.
2. Click **Deploy module** to initialize the asset in the system.
3. The system will automatically redirect you to the **Curriculum Editor**.

### Step 5: Build Content Architecture
1. **Add Sections:** Create the skeletal structure of your course.
2. **Add Lessons:** Populate sections with video lessons, documents, or quizzes.
3. **AI Generation:** Use the **AI Asset Generator** to build quizzes from transcripts or generate curriculum structures if needed.

### Step 6: Final Review & Publishing
1. Ensure all lessons have content.
2. Preview the course as a student to verify the user experience.
3. Set the status to **Published** to make it live in the library.
